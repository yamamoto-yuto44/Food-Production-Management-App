# 食品製造管理アプリ

## 📌 概要
食品製造現場の実務を想定した在庫・生産管理アプリケーションです。

当日の生産登録を行うと、レシピに基づき、原料在庫が自動で減少し、在庫履歴として記録されます。

管理者は原料の発注・レシピ管理を行うことができます。

---

## 🎯 開発目的

Spring Boot / JPA / Spring Security の理解を深めるため、
業務フローを意識したアプリケーションを設計・開発しました。

特に以下を意識しました：

- 生産登録 → 在庫減少 → 履歴保存 の業務フロー設計
- ロール別アクセス制御（ADMIN / USER）
- UIの統一（Bootstrap + レイアウト分離）
- DTOを用いた責務分離

| 機能 | USER | ADMIN |

| 原料作成・削除 | × | ○ |

---

## 📊 主な機能

### ① 生産登録
- レシピに基づいて原料在庫を自動減少
- 在庫履歴テーブルに記録

### ② 原料在庫管理
- 在庫一覧表示
- 発注による在庫増加
- 在庫履歴確認

### ③ レシピ管理
- 製品ごとの使用原料と使用量を設定

### ④ 認証・認可
- Spring Securityによるログイン機能
- ロールベースアクセス制御
- 403ページ実装

---

## 🧠 設計上の工夫

- Service層で在庫減少処理を一元管理
- DTOを用いてViewとEntityを分離
- ロールによる画面表示制御
- URL直打ち対策としてSecurityConfigで制御
