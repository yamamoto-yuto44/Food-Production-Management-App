# 食品製造管理アプリ

## 概要
食品製造現場の実務を想定した在庫・生産管理アプリケーションです。

当日の生産登録を行うと、レシピに基づき、原料在庫が自動で減少し、生産履歴・在庫履歴として記録されます。

また、原料発注を行うことで、入荷処理によって在庫を補充することができます。

ユーザー権限（USER / ADMIN）による機能制御も実装しています。

## 権限制御

| 機能 | USER | ADMIN |
|------|------|-------|
| 生産記録の登録 | ○ | ○ |
| 生産履歴の確認・修正 | ○ | ○ |
| 原料在庫の確認 | ○ | ○ |
| レシピ一覧の閲覧 | ○ | ○ |
| 原料の発注 | × | ○ |
| レシピの登録 | × | ○ |
| レシピの削除 | × | ○ |
| 原料の登録 | × | ○ |

---

## 主な機能

### ① 生産登録
- レシピに基づいて原料在庫を自動減少
- 在庫履歴テーブルに記録

### ② 原料在庫管理
- 在庫一覧表示
- 発注による在庫増加
- 在庫履歴確認

### ③ レシピ管理
- 製品ごとの使用原料と使用量を設定

### ④ 認証・認可
- Spring Securityによるログイン機能
- ロールベースアクセス制御
- 403ページ実装

---

## 設計上の工夫

- Service層で在庫減少処理を一元管理
- DTOを用いてViewとEntityを分離
- ロールによる画面表示制御
- URL直打ち対策としてSecurityConfigで制御
